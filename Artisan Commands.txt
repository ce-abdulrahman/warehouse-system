Here is the complete set of terminal commands to generate the entire folder and file structure for your application.

Copy and paste these blocks into your terminal (Git Bash, Terminal, or VS Code terminal) inside your project root.

ðŸ”· Part 1: Backend Structure (Artisan Commands)
This generates the Models, Migrations, Controllers, and Form Requests.

Bash

# 1. Settings Module
php artisan make:model Setting -mcr
php artisan make:request UpdateSettingRequest

# 2. Item Module
php artisan make:model Item -mcr
php artisan make:request StoreItemRequest
php artisan make:request UpdateItemRequest

# 3. Warehouse Module
php artisan make:model Warehouse -mcr
php artisan make:request StoreWarehouseRequest
php artisan make:request UpdateWarehouseRequest

# 4. Supplier Module
php artisan make:model Supplier -mcr
php artisan make:request StoreSupplierRequest
php artisan make:request UpdateSupplierRequest

# 5. Stock Movement Module
# Note: We use -mc (Migration+Controller) because we don't need a full resource for movements (usually just index/create)
php artisan make:model StockMovement -mc
php artisan make:request StoreMovementRequest

# 6. User/Profile Extensions (if not using Breeze's default)
php artisan make:controller ProfileController
ðŸ”· Part 2: Frontend Structure (Views)
This uses standard shell commands to create the directories and empty Blade files.

Note: If you are on Windows Command Prompt (cmd), replace touch with type nul >. If you are using Git Bash, VS Code Terminal, or Mac/Linux, the commands below work as is.

Bash

# 1. Create Directories
mkdir -p resources/views/layouts
mkdir -p resources/views/settings
mkdir -p resources/views/items
mkdir -p resources/views/warehouses
mkdir -p resources/views/suppliers
mkdir -p resources/views/movements
mkdir -p resources/views/profile

# 2. Create Layout Files (Overwriting default if exists)
touch resources/views/layouts/app.blade.php
touch resources/views/layouts/navigation.blade.php
touch resources/views/dashboard.blade.php

# 3. Create Settings Views
touch resources/views/settings/index.blade.php

# 4. Create Item Views
touch resources/views/items/index.blade.php
touch resources/views/items/create.blade.php
touch resources/views/items/edit.blade.php
touch resources/views/items/show.blade.php

# 5. Create Warehouse Views
touch resources/views/warehouses/index.blade.php
touch resources/views/warehouses/create.blade.php
touch resources/views/warehouses/edit.blade.php

# 6. Create Supplier Views
touch resources/views/suppliers/index.blade.php
touch resources/views/suppliers/create.blade.php
touch resources/views/suppliers/edit.blade.php

# 7. Create Movement Views
touch resources/views/movements/index.blade.php
touch resources/views/movements/create.blade.php

# 8. Create Profile Views
touch resources/views/profile/edit.blade.php
